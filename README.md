# Chemistry Code
I've always found that I understand/remember concepts better if I write code capable of doing it, so I wrote this code for the last few chapters of "Discovering Design With Chemistry".

## Notes
Don't use this to actually do your work. Here's a few reasons why:
 - If you use this code, you will probably end up not understanding the concepts behind it.
 - I can't garantee that I did things correctly -- I personally found chemistry quite painful.
 - The code here does NOT handle significant figures at all.
